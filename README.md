# 통계 처리 패키지 `statkit`

## 폴더 구조
```
statkit/                # 통계 기능 모듈 패키지
   ├─ __init__.py          # 공개 API를 한글 함수 이름으로 정리
   ├─ descriptive.py       # 평균, 중앙값, 분산 등 기술 통계 함수
   └─ inference.py         # 신뢰구간 등 간단한 추론 통계 함수
```

## 설치 및 사용
1. 현재 디렉터리를 파이썬 패키지 경로에 추가합니다.
   ```bash
   export PYTHONPATH="$PWD:$PYTHONPATH"
   ```
2. 파이썬에서 모듈을 임포트하여 사용합니다.
   ```python
   from statkit import 평균, 요약, 표본평균_신뢰구간

   데이터 = [1, 2, 3, 4, 5]
   print(평균(데이터))
   print(요약(데이터))
   print(표본평균_신뢰구간(데이터, 신뢰수준=0.95))
   ```

## 지원 기능
- 평균, 중앙값, 분산, 표준편차 계산
- 데이터의 기본 요약 통계량 제공
- 정규분포 가정 하에서 표본평균 신뢰구간 계산 (Z-값 기반)
